## 功能

- 📦 自动推荐英雄出装
- 🎉 弹出天赋列表并一键导入
- ✨ 国际化支持
    - 🇨🇳 `简体中文`
    - 🇺🇸 `英语`
    - 🇫🇷 `法语`
- ℹ️ 更新提示
- 😎 针对不同的模式 **召唤师峡谷** / **极地大乱斗** / **无限火力** 提供多种数据源
    - `op.gg`
    - `lolalytics.com`
    - `u.gg`
    - `champion.gg`
    - `murderbridge.com`
    - `101.qq.com`
    - ...

## 使用方法

### 导入出装数据

1. 右键以 **管理员** 身份运行 ChampR
2. 选择英雄联盟的 **安装目录**
3. 选择数据源
4. 单击 `导入` 按钮
5.在游戏内商店界面即可选择出装

### Apply Runes

1. 右键以 **管理员** 身份运行 ChampR
1. 选择英雄联盟的 **安装目录**
1. 游戏内选择英雄时会弹出天赋窗口
1. 点击 `应用` 图标

## 开发

### 环境要求

- [Node.js](https://nodejs.org/en/) >= 14

### 安装依赖

```console
pnpm install
```

### 运行

```console
pnpm run start
```

### 编译

```console
pnpm run build:local
```
