## Development

### Prerequisite

- [Node.js](https://nodejs.org/en/) >= 10
- [yarn](https://classic.yarnpkg.com/lang/en/)

### Install dependencies

```bash
yarn
```

### Start

```bash
yarn start
```

### Build

```bash
yarn build
```
